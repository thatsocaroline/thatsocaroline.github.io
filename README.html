Business, Management, and Information Technology

Different courses

  • Digital Information Technology

    Develop basic technology skills and understand cyber security.

  • Business and Entrepreneurship Principles

    Teaches students business plan development.

  • Financial Accounting with Excel

    Develop financial skills and use Microsoft Excel for business records.

  • Legal Aspects of Business

    Business law concepts, forms of business ownership, insurance, governmental regulations, management systems, and human resources.

  • Business Analysis

    Business systems, accounting, skills in data analysis, and business information management.

  • Diversified Career Technology (DCT-OJT)

    Students have the opportunity to work at local businesses

  • Foundations of Programming

    Use HTML, CSS, and JavaScript to learn the basics of coding (how to create a website, games, and applications.)

  • Procedural Programming

    Learn software development using the C# and C++ programming languages. Students will also use SQL to learn about databases.

  • Object Oriented Programming Essentials

    Learn how to program using the Python programming language.

  • AP Computer Science Principles

    Discover the large basic ideas of computing using the JavaScript programming language.

  • AP Computer Science A

    Builds on the principles of AP Computer Science and uses Python for advanced programming.
